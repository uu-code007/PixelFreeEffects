## 2.4.18
- Android SDK 更新至 v2.4.18
- 新增美妆功能支持
  - 支持设置美妆路径 (`setMakeupPath`)
  - 支持清除美妆 (`clearMakeup`)
  - 支持设置美妆部位程度 (`setMakeupPartDegree`)
- 新增人脸检测相关接口
  - 获取人脸矩形 (`getFaceRect`)
  - 获取人脸数量 (`getFaceSize`)
  - 设置检测模式 (`setDetectMode`)
  - 检测是否有人脸 (`hasFace`)
- 新增版本查询接口 (`getVersion`)
- 新增日志级别设置接口 (`setVLogLevel`)
- 修复 Android API 兼容性问题
- 完善所有 C 接口的 Flutter 封装
