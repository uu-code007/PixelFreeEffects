# pixelFree 商业级美颜SDK

[English Version](./README_en.md)

#### 项目介绍

**pixelFree** 是基于个人几年对图形学习，总结开发的SDK, 轻量级高性能，主要用于直播，短视频，证件照.....

主要功能包含：美白，红润，磨皮，锐化，大眼，瘦脸…….

集成接入参考  *pixelFreeEffects* 演示**DEMO**
