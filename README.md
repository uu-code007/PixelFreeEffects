# pixelFree 商业级美颜SDK （全平台支持）

[English Version](./README_en.md)

#### 项目介绍

**pixelFree** 是一款支持全平台, 轻量级高性能的美颜特效 SDK，主要用于直播，短视频，证件照.....

主要功能包含：美白，红润，磨皮，锐化，大眼，瘦脸,2D 贴纸 …….

集成接入参考  *pixelFreeEffects* 演示**DEMO**
